# Hasta Mobil Uygulaması (React Native / Expo)

## Kurulum
```bash
cp .env.example .env
npm install
npm run start
```

API adresi: `EXPO_PUBLIC_API_BASE_URL=http://localhost:8000/api/v1`

## Ekranlar
- Giriş / Kayıt / Şifremi Unuttum
- Ana Sayfa
- Bugünkü Plan
- Egzersiz Detayı
- Kamera Takibi
- İlerleme
- Ağrı Günlüğü
- Bildirimler
- Profil ve Ayarlar
- Erişilebilirlik
